# Weather Forecast API Tool

## Overview
A Python application that retrieves real-time weather data from a public API and displays temperature, humidity, and forecast information in a clean, user-friendly format.

## Features
- Fetches live weather data using HTTP requests
- Parses JSON responses
- Displays current conditions and multi-day forecasts
- Simple, readable output for quick use

## Technologies Used
- Python
- Requests (API calls)
- JSON Parsing

## How It Works
The program sends a request to a weather API endpoint, receives JSON data, extracts relevant fields, and prints them in a structured format.
